# Projetos Complementares

* [sigma-rust](https://github.com/Yamato-Security/sigma-rust/) - nosso repositório bifurcado do crate [sigma-rust](https://github.com/jopohl/sigma-rust) com atualizações como suporte a regras de correlação, etc...
* [suzaku-rules](https://github.com/Yamato-Security/suzaku-rules) - nosso repositório de regras Sigma. Novas regras Sigma upstream são adicionadas automaticamente. Também hospedamos nossas próprias regras aqui.
* [suzaku-sample-data](https://github.com/Yamato-Security/suzaku-sample-data) - um repositório de diversos logs de nuvem com evidências de ataque dentro deles, usado para testar o Suzaku, bem como para escrever novas regras de detecção.
