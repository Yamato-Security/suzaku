# Proyek Pendamping

* [sigma-rust](https://github.com/Yamato-Security/sigma-rust/) - repositori hasil fork kami dari crate [sigma-rust](https://github.com/jopohl/sigma-rust) dengan pembaruan seperti dukungan untuk aturan korelasi, dll...
* [suzaku-rules](https://github.com/Yamato-Security/suzaku-rules) - repositori aturan Sigma milik kami. Aturan Sigma upstream yang baru ditambahkan secara otomatis. Kami juga menyimpan aturan kami sendiri di sini.
* [suzaku-sample-data](https://github.com/Yamato-Security/suzaku-sample-data) - repositori berisi berbagai log cloud dengan bukti serangan di dalamnya yang digunakan untuk menguji Suzaku serta untuk menulis aturan deteksi baru.
