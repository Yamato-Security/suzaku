# Супутні проєкти

* [sigma-rust](https://github.com/Yamato-Security/sigma-rust/) - наш форк репозиторію крейту [sigma-rust](https://github.com/jopohl/sigma-rust) з оновленнями, такими як підтримка кореляційних правил тощо...
* [suzaku-rules](https://github.com/Yamato-Security/suzaku-rules) - наш репозиторій правил Sigma. Нові вихідні правила Sigma додаються автоматично. Тут ми також розміщуємо власні правила.
* [suzaku-sample-data](https://github.com/Yamato-Security/suzaku-sample-data) - репозиторій з різноманітними хмарними журналами, що містять докази атак, який використовується для тестування Suzaku, а також для написання нових правил виявлення.
