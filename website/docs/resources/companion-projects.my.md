# တွဲဖက်စီမံကိန်းများ (Companion Projects)

* [sigma-rust](https://github.com/Yamato-Security/sigma-rust/) - [sigma-rust](https://github.com/jopohl/sigma-rust) crate ၏ ကျွန်ုပ်တို့ fork ပြုလုပ်ထားသော repository ဖြစ်ပြီး correlation rules အတွက် ပံ့ပိုးမှု စသည်ဖြင့် အပ်ဒိတ်များ ပါဝင်သည်။
* [suzaku-rules](https://github.com/Yamato-Security/suzaku-rules) - ကျွန်ုပ်တို့၏ Sigma rules repository ဖြစ်သည်။ အသစ်ထွက်ရှိသော upstream Sigma rules များကို အလိုအလျောက် ထည့်သွင်းပေးပါသည်။ ကျွန်ုပ်တို့၏ ကိုယ်ပိုင် rules များကိုလည်း ဤနေရာတွင် ထားရှိပါသည်။
* [suzaku-sample-data](https://github.com/Yamato-Security/suzaku-sample-data) - Suzaku ကို စမ်းသပ်ရန်အတွက်နှင့် detection rules အသစ်များ ရေးသားရန်အတွက် အသုံးပြုသော တိုက်ခိုက်မှု သက်သေအထောက်အထားများ ပါဝင်သည့် cloud log အမျိုးမျိုးကို သိမ်းဆည်းထားသော repository တစ်ခု ဖြစ်သည်။
