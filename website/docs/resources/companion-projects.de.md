# Begleitprojekte

* [sigma-rust](https://github.com/Yamato-Security/sigma-rust/) - unser geforktes Repository des [sigma-rust](https://github.com/jopohl/sigma-rust)-Crates mit Aktualisierungen wie der Unterstützung für Korrelationsregeln usw...
* [suzaku-rules](https://github.com/Yamato-Security/suzaku-rules) - unser Repository mit Sigma-Regeln. Neue Upstream-Sigma-Regeln werden automatisch hinzugefügt. Wir hosten hier auch unsere eigenen Regeln.
* [suzaku-sample-data](https://github.com/Yamato-Security/suzaku-sample-data) - ein Repository mit verschiedenen Cloud-Logs, die Angriffsspuren enthalten und zum Testen von Suzaku sowie zum Schreiben neuer Erkennungsregeln verwendet werden.
