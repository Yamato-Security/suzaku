# Contribuciones y soporte

## Contribución

Nos encantaría recibir cualquier forma de contribución.
Las pull requests, la creación de reglas y los registros de ejemplo son lo mejor, pero las solicitudes de funciones, la notificación de errores, etc... también son muy bienvenidas.

Como mínimo, **si te gustan nuestras herramientas y recursos, ¡danos una estrella en GitHub y muéstranos tu apoyo!**

## Envío de errores

* Por favor, envía cualquier error que encuentres [aquí.](https://github.com/Yamato-Security/suzaku/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D)
* Este proyecto se mantiene actualmente de forma activa y estaremos encantados de corregir cualquier error reportado.
* Si encuentras algún problema (falsos positivos, errores, etc...) con las reglas de Suzaku, por favor repórtalo en la página de issues de GitHub de suzaku-rules [aquí](https://github.com/Yamato-Security/suzaku-rules/issues/new).
* Si encuentras algún problema (falsos positivos, errores, etc...) con las reglas de Sigma, por favor repórtalo en la página de issues de GitHub de SigmaHQ original [aquí](https://github.com/SigmaHQ/sigma/issues).
