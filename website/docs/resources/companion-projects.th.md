# โปรเจกต์ที่เกี่ยวข้อง

* [sigma-rust](https://github.com/Yamato-Security/sigma-rust/) - repository ที่เรา fork มาจาก crate [sigma-rust](https://github.com/jopohl/sigma-rust) พร้อมการอัปเดตต่าง ๆ เช่น การรองรับ correlation rules เป็นต้น...
* [suzaku-rules](https://github.com/Yamato-Security/suzaku-rules) - repository ของกฎ Sigma ของเรา กฎ Sigma ใหม่จาก upstream จะถูกเพิ่มเข้ามาโดยอัตโนมัติ เรายังโฮสต์กฎที่เราเขียนเองไว้ที่นี่อีกด้วย
* [suzaku-sample-data](https://github.com/Yamato-Security/suzaku-sample-data) - repository ของ cloud log หลากหลายประเภทที่มีหลักฐานการโจมตีอยู่ภายใน ใช้สำหรับทดสอบ Suzaku รวมถึงสำหรับการเขียนกฎตรวจจับใหม่
