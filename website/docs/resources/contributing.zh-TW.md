# 貢獻與支援

## 貢獻

我們非常歡迎任何形式的貢獻。
Pull request、規則建立與範例日誌是最棒的貢獻，但功能請求、回報錯誤等等也非常歡迎。

至少，**如果您喜歡我們的工具與資源，請在 GitHub 上給我們一顆星並表達您的支持！**

## 錯誤回報

* 請在[此處](https://github.com/Yamato-Security/suzaku/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D)回報您發現的任何錯誤。
* 本專案目前正積極維護中，我們很樂意修復任何回報的錯誤。
* 如果您發現 Suzaku 規則有任何問題（誤報、錯誤等等），請至 suzaku-rules GitHub issues 頁面[此處](https://github.com/Yamato-Security/suzaku-rules/issues/new)回報。
* 如果您發現 Sigma 規則有任何問題（誤報、錯誤等等），請至上游 SigmaHQ GitHub issues 頁面[此處](https://github.com/SigmaHQ/sigma/issues)回報。

## 授權

* Suzaku 以 [AGPLv3](https://www.gnu.org/licenses/agpl-3.0.en.html) 授權發布，所有規則則以 [Detection Rule License (DRL) 1.1](https://github.com/SigmaHQ/sigma/blob/master/LICENSE.Detection.Rules.md) 授權發布。
* 您可以自由地在內部、SaaS 解決方案、顧問工作等場合使用 Suzaku。
然而，如果您對其進行了改進，我們請求您將這些改進開源並回饋給專案。
