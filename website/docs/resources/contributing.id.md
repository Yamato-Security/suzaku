# Kontribusi & Dukungan

## Kontribusi

Kami akan sangat senang menerima segala bentuk kontribusi.
Pull request, pembuatan aturan, dan contoh log adalah yang terbaik, tetapi permintaan fitur, pemberitahuan bug, dll... juga sangat kami hargai.

Setidaknya, **jika Anda menyukai alat dan sumber daya kami, mohon berikan kami bintang di GitHub dan tunjukkan dukungan Anda!**

## Pengiriman Bug

* Mohon kirimkan bug apa pun yang Anda temukan [di sini.](https://github.com/Yamato-Security/suzaku/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D)
* Proyek ini saat ini dipelihara secara aktif dan kami dengan senang hati memperbaiki bug apa pun yang dilaporkan.
* Jika Anda menemukan masalah apa pun (positif palsu, bug, dll...) dengan aturan Suzaku, mohon laporkan ke halaman issues GitHub suzaku-rules [di sini](https://github.com/Yamato-Security/suzaku-rules/issues/new).
* Jika Anda menemukan masalah apa pun (positif palsu, bug, dll...) dengan aturan Sigma, mohon laporkan ke halaman issues GitHub SigmaHQ hulu [di sini](https://github.com/SigmaHQ/sigma/issues).

## Lisensi

* Suzaku dirilis di bawah [AGPLv3](https://www.gnu.org/licenses/agpl-3.0.en.html) dan semua aturan dirilis di bawah [Detection Rule License (DRL) 1.1](https://github.com/SigmaHQ/sigma/blob/master/LICENSE.Detection.Rules.md).
* Anda dapat dengan bebas menggunakan Suzaku secara internal, untuk solusi SaaS, untuk pekerjaan konsultasi, dll...
Namun, jika Anda melakukan perbaikan terhadapnya, kami meminta Anda untuk membuka kode sumber perbaikan tersebut dan memberikannya kembali kepada proyek.
