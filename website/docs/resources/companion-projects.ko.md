# 관련 프로젝트

* [sigma-rust](https://github.com/Yamato-Security/sigma-rust/) - 상관 관계 규칙 지원 등의 업데이트가 적용된 [sigma-rust](https://github.com/jopohl/sigma-rust) 크레이트의 포크 저장소입니다...
* [suzaku-rules](https://github.com/Yamato-Security/suzaku-rules) - Sigma 규칙을 모아 둔 저장소입니다. 새로운 업스트림 Sigma 규칙이 자동으로 추가됩니다. 또한 자체 규칙도 여기에서 호스팅합니다.
* [suzaku-sample-data](https://github.com/Yamato-Security/suzaku-sample-data) - Suzaku 테스트와 새로운 탐지 규칙 작성에 사용되는, 공격 증거가 포함된 다양한 클라우드 로그를 모아 둔 저장소입니다.
