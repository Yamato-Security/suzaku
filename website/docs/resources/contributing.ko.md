# 기여 및 지원

## 기여

어떤 형태의 기여든 환영합니다.
풀 리퀘스트, 룰 작성, 샘플 로그가 가장 좋지만 기능 요청, 버그 알림 등도 매우 환영합니다.

적어도, **저희 도구와 리소스가 마음에 드신다면 GitHub에서 별을 눌러 응원을 보내주세요!**

## 버그 제보

* 발견하신 버그는 [여기에서](https://github.com/Yamato-Security/suzaku/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D) 제보해 주세요.
* 이 프로젝트는 현재 활발히 유지보수되고 있으며, 제보된 버그를 기꺼이 수정합니다.
* Suzaku 룰에서 문제(오탐, 버그 등)를 발견하시면 suzaku-rules GitHub 이슈 페이지 [여기에](https://github.com/Yamato-Security/suzaku-rules/issues/new) 제보해 주세요.
* Sigma 룰에서 문제(오탐, 버그 등)를 발견하시면 상위 SigmaHQ GitHub 이슈 페이지 [여기에](https://github.com/SigmaHQ/sigma/issues) 제보해 주세요.

## 라이선스

* Suzaku는 [AGPLv3](https://www.gnu.org/licenses/agpl-3.0.en.html)로 배포되며, 모든 룰은 [Detection Rule License (DRL) 1.1](https://github.com/SigmaHQ/sigma/blob/master/LICENSE.Detection.Rules.md)로 배포됩니다.
* Suzaku를 내부 용도, SaaS 솔루션, 컨설팅 업무 등으로 자유롭게 사용하실 수 있습니다.
다만, 개선 사항을 만드신다면 그 개선 사항을 오픈소스로 공개하여 프로젝트에 환원해 주시기를 부탁드립니다.
