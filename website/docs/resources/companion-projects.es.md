# Proyectos complementarios

* [sigma-rust](https://github.com/Yamato-Security/sigma-rust/) - nuestro repositorio bifurcado del crate [sigma-rust](https://github.com/jopohl/sigma-rust) con actualizaciones como soporte para reglas de correlación, etc...
* [suzaku-rules](https://github.com/Yamato-Security/suzaku-rules) - nuestro repositorio de reglas Sigma. Las nuevas reglas Sigma upstream se agregan automáticamente. También alojamos nuestras propias reglas aquí.
* [suzaku-sample-data](https://github.com/Yamato-Security/suzaku-sample-data) - un repositorio de varios registros en la nube con evidencia de ataques en su interior, utilizado para probar Suzaku así como para escribir nuevas reglas de detección.
