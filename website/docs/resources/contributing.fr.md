# Contribution et support

## Contribution

Nous serions ravis de recevoir toute forme de contribution.
Les pull requests, la création de règles et les exemples de logs sont les bienvenus, mais les demandes de fonctionnalités, les signalements de bugs, etc. sont également très appréciés.

Au minimum, **si vous appréciez nos outils et nos ressources, merci de nous attribuer une étoile sur GitHub et de manifester votre soutien !**

## Signalement de bugs

* Veuillez signaler tout bug que vous trouvez [ici.](https://github.com/Yamato-Security/suzaku/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D)
* Ce projet est actuellement activement maintenu et nous sommes heureux de corriger tout bug signalé.
* Si vous trouvez des problèmes (faux positifs, bugs, etc.) avec les règles Suzaku, veuillez les signaler sur la page des issues GitHub de suzaku-rules [ici](https://github.com/Yamato-Security/suzaku-rules/issues/new).
* Si vous trouvez des problèmes (faux positifs, bugs, etc.) avec les règles Sigma, veuillez les signaler sur la page des issues GitHub en amont de SigmaHQ [ici](https://github.com/SigmaHQ/sigma/issues).
