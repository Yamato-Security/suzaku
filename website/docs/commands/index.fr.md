# Liste des commandes

## Commandes d'analyse :
* `aws-ct-metrics` : Génère des métriques à partir des journaux AWS CloudTrail
  
## Commandes de synthèse :
* `aws-ct-summary` : Génère une synthèse de l'activité à partir des ARN utilisateur

## Commandes de chronologie DFIR :
* `aws-ct-timeline` : Crée une chronologie DFIR AWS CloudTrail
* `update-rules` : Met à jour les règles de détection Sigma

## Commandes générales :
* `help` : Affiche le menu d'aide des commandes
