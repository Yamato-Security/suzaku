# Komut Listesi

## Analiz Komutları:
* `aws-ct-metrics`: AWS CloudTrail loglarından metrikler üretir
  
## Özet Komutları:
* `aws-ct-summary`: Kullanıcı ARN'lerinden bir etkinlik özeti üretir

## DFIR Zaman Çizelgesi Komutları:
* `aws-ct-timeline`: Bir AWS CloudTrail DFIR zaman çizelgesi oluşturur
* `update-rules`: Sigma tespit kurallarını günceller

## Genel Komutlar:
* `help`: Komutlar için yardım menüsünü yazdırır
