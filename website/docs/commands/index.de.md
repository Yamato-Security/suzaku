# Befehlsliste

## Analysebefehle:
* `aws-ct-metrics`: Erzeugt Metriken aus AWS-CloudTrail-Logs
  
## Zusammenfassungsbefehle:
* `aws-ct-summary`: Erzeugt eine Zusammenfassung der Aktivitäten aus Benutzer-ARNs

## DFIR-Zeitleisten-Befehle:
* `aws-ct-timeline`: Erstellt eine AWS-CloudTrail-DFIR-Zeitleiste
* `update-rules`: Aktualisiert die Sigma-Erkennungsregeln

## Allgemeine Befehle:
* `help`: Gibt das Hilfemenü für Befehle aus
