# ကွန်မန်းစာရင်း

## ခွဲခြမ်းစိတ်ဖြာမှု ကွန်မန်းများ:
* `aws-ct-metrics`: AWS CloudTrail မှတ်တမ်းများမှ မက်ထရစ်များ ထုတ်လုပ်သည်
  
## အကျဉ်းချုပ် ကွန်မန်းများ:
* `aws-ct-summary`: အသုံးပြုသူ ARN များမှ လှုပ်ရှားမှု အကျဉ်းချုပ်ကို ထုတ်လုပ်သည်

## DFIR အချိန်ဇယား ကွန်မန်းများ:
* `aws-ct-timeline`: AWS CloudTrail DFIR အချိန်ဇယားကို ဖန်တီးသည်
* `update-rules`: Sigma ထောက်လှမ်းရှာဖွေမှု စည်းမျဉ်းများကို မွမ်းမံသည်

## အထွေထွေ ကွန်မန်းများ:
* `help`: ကွန်မန်းများအတွက် အကူအညီ မီနူးကို ပုံနှိပ်ထုတ်သည်
