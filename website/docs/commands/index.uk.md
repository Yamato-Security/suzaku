# Список команд

## Команди аналізу:
* `aws-ct-metrics`: Генерує метрики з журналів AWS CloudTrail
  
## Команди підсумків:
* `aws-ct-summary`: Генерує підсумок активності за ARN користувачів

## Команди часової шкали DFIR:
* `aws-ct-timeline`: Створює часову шкалу DFIR для AWS CloudTrail
* `update-rules`: Оновлює правила виявлення Sigma

## Загальні команди:
* `help`: Виводить меню довідки для команд
