# कमांड सूची

## विश्लेषण कमांड:
* `aws-ct-metrics`: AWS CloudTrail लॉग से मेट्रिक्स उत्पन्न करता है
  
## सारांश कमांड:
* `aws-ct-summary`: उपयोगकर्ता ARN से गतिविधि का सारांश उत्पन्न करता है

## DFIR टाइमलाइन कमांड:
* `aws-ct-timeline`: एक AWS CloudTrail DFIR टाइमलाइन बनाता है
* `update-rules`: Sigma डिटेक्शन नियमों को अपडेट करता है

## सामान्य कमांड:
* `help`: कमांड के लिए सहायता मेन्यू प्रिंट करता है
