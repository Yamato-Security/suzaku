# รายการคำสั่ง

## คำสั่งสำหรับการวิเคราะห์:
* `aws-ct-metrics`: สร้างเมตริกจากบันทึก AWS CloudTrail

## คำสั่งสำหรับการสรุป:
* `aws-ct-summary`: สร้างสรุปกิจกรรมจาก user ARN

## คำสั่งสำหรับไทม์ไลน์ DFIR:
* `aws-ct-timeline`: สร้างไทม์ไลน์ DFIR ของ AWS CloudTrail
* `update-rules`: อัปเดตกฎการตรวจจับของ Sigma

## คำสั่งทั่วไป:
* `help`: แสดงเมนูช่วยเหลือสำหรับคำสั่งต่าง ๆ
