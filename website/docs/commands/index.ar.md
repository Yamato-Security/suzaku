# قائمة الأوامر

## أوامر التحليل:
* `aws-ct-metrics`: يُنشئ مقاييس من سجلات AWS CloudTrail
  
## أوامر الملخص:
* `aws-ct-summary`: يُنشئ ملخصًا للنشاط من معرّفات ARN الخاصة بالمستخدمين

## أوامر الجدول الزمني لـ DFIR:
* `aws-ct-timeline`: يُنشئ جدولًا زمنيًا لـ DFIR من AWS CloudTrail
* `update-rules`: يُحدّث قواعد الكشف الخاصة بـ Sigma

## الأوامر العامة:
* `help`: يطبع قائمة المساعدة للأوامر
