# Daftar Perintah

## Perintah Analisis:
* `aws-ct-metrics`: Menghasilkan metrik dari log AWS CloudTrail
  
## Perintah Ringkasan:
* `aws-ct-summary`: Menghasilkan ringkasan aktivitas dari ARN pengguna

## Perintah Timeline DFIR:
* `aws-ct-timeline`: Membuat timeline DFIR AWS CloudTrail
* `update-rules`: Memperbarui aturan deteksi Sigma

## Perintah Umum:
* `help`: Menampilkan menu bantuan untuk perintah
