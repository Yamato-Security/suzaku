# Lista de comandos

## Comandos de análisis:
* `aws-ct-metrics`: Genera métricas a partir de los registros de AWS CloudTrail
  
## Comandos de resumen:
* `aws-ct-summary`: Genera un resumen de la actividad a partir de los ARN de usuario

## Comandos de cronología DFIR:
* `aws-ct-timeline`: Crea una cronología DFIR de AWS CloudTrail
* `update-rules`: Actualiza las reglas de detección de Sigma

## Comandos generales:
* `help`: Muestra el menú de ayuda de los comandos
