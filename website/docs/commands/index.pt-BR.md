# Lista de Comandos

## Comandos de Análise:
* `aws-ct-metrics`: Gera métricas a partir de logs do AWS CloudTrail
  
## Comandos de Resumo:
* `aws-ct-summary`: Gera um resumo de atividade a partir de ARNs de usuário

## Comandos de Linha do Tempo DFIR:
* `aws-ct-timeline`: Cria uma linha do tempo DFIR do AWS CloudTrail
* `update-rules`: Atualiza as regras de detecção do Sigma

## Comandos Gerais:
* `help`: Exibe o menu de ajuda dos comandos
