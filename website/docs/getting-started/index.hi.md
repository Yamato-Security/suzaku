# डाउनलोड

कृपया Suzaku का नवीनतम स्थिर संस्करण कंपाइल की गई बाइनरी के साथ डाउनलोड करें या [Releases](https://github.com/Yamato-Security/suzaku/releases) पृष्ठ से स्रोत कोड को कंपाइल करें।

हम निम्नलिखित आर्किटेक्चर के लिए बाइनरी प्रदान करते हैं:
- Linux ARM 64-bit GNU (`suzaku-x.x.x-lin-aarch64-gnu`)
- Linux Intel 64-bit GNU (`suzaku-x.x.x-lin-x64-gnu`)
- Linux Intel 64-bit MUSL (`suzaku-x.x.x-lin-x64-musl`)
- macOS ARM 64-bit (`suzaku-x.x.x-mac-aarch64`)
- macOS Intel 64-bit (`suzaku-x.x.x-mac-x64`)
- Windows ARM 64-bit (`suzaku-x.x.x-win-aarch64.exe`)
- Windows Intel 64-bit (`suzaku-x.x.x-win-x64.exe`)
- Windows Intel 32-bit (`suzaku-x.x.x-win-x86.exe`)

> [किसी कारणवश Linux ARM MUSL बाइनरी ठीक से नहीं चलती](https://github.com/Yamato-Security/hayabusa/issues/1332) इसलिए हम वह बाइनरी प्रदान नहीं करते। यह हमारे नियंत्रण से बाहर है, इसलिए जब यह ठीक हो जाएगा तब भविष्य में हम इसे प्रदान करने की योजना बना रहे हैं।
