# İndirmeler

Lütfen derlenmiş ikili dosyalarla birlikte Suzaku'nun en son kararlı sürümünü indirin veya [Releases](https://github.com/Yamato-Security/suzaku/releases) sayfasından kaynak kodunu derleyin.

Aşağıdaki mimariler için ikili dosyalar sağlıyoruz:
- Linux ARM 64-bit GNU (`suzaku-x.x.x-lin-aarch64-gnu`)
- Linux Intel 64-bit GNU (`suzaku-x.x.x-lin-x64-gnu`)
- Linux Intel 64-bit MUSL (`suzaku-x.x.x-lin-x64-musl`)
- macOS ARM 64-bit (`suzaku-x.x.x-mac-aarch64`)
- macOS Intel 64-bit (`suzaku-x.x.x-mac-x64`)
- Windows ARM 64-bit (`suzaku-x.x.x-win-aarch64.exe`)
- Windows Intel 64-bit (`suzaku-x.x.x-win-x64.exe`)
- Windows Intel 32-bit (`suzaku-x.x.x-win-x86.exe`)

> [Bir nedenden dolayı Linux ARM MUSL ikili dosyası düzgün çalışmıyor](https://github.com/Yamato-Security/hayabusa/issues/1332), bu yüzden o ikili dosyayı sağlamıyoruz. Bu bizim kontrolümüz dışında olduğundan, sorun giderildiğinde gelecekte sağlamayı planlıyoruz.
