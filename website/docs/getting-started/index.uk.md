# Завантаження

Будь ласка, завантажте останню стабільну версію Suzaku зі скомпільованими бінарними файлами або скомпілюйте вихідний код зі сторінки [Releases](https://github.com/Yamato-Security/suzaku/releases).

Ми надаємо бінарні файли для таких архітектур:
- Linux ARM 64-bit GNU (`suzaku-x.x.x-lin-aarch64-gnu`)
- Linux Intel 64-bit GNU (`suzaku-x.x.x-lin-x64-gnu`)
- Linux Intel 64-bit MUSL (`suzaku-x.x.x-lin-x64-musl`)
- macOS ARM 64-bit (`suzaku-x.x.x-mac-aarch64`)
- macOS Intel 64-bit (`suzaku-x.x.x-mac-x64`)
- Windows ARM 64-bit (`suzaku-x.x.x-win-aarch64.exe`)
- Windows Intel 64-bit (`suzaku-x.x.x-win-x64.exe`)
- Windows Intel 32-bit (`suzaku-x.x.x-win-x86.exe`)

> [З певних причин бінарний файл Linux ARM MUSL не запускається належним чином](https://github.com/Yamato-Security/hayabusa/issues/1332), тому ми не надаємо цей бінарний файл. Це поза нашим контролем, тож ми плануємо надати його в майбутньому, коли проблему буде виправлено.
