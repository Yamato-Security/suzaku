# التنزيلات

يرجى تنزيل أحدث إصدار مستقر من Suzaku مع الملفات الثنائية المُجمَّعة أو تجميع الكود المصدري من صفحة [الإصدارات](https://github.com/Yamato-Security/suzaku/releases).

نوفّر ملفات ثنائية للبُنى التالية:
- Linux ARM 64-bit GNU (`suzaku-x.x.x-lin-aarch64-gnu`)
- Linux Intel 64-bit GNU (`suzaku-x.x.x-lin-x64-gnu`)
- Linux Intel 64-bit MUSL (`suzaku-x.x.x-lin-x64-musl`)
- macOS ARM 64-bit (`suzaku-x.x.x-mac-aarch64`)
- macOS Intel 64-bit (`suzaku-x.x.x-mac-x64`)
- Windows ARM 64-bit (`suzaku-x.x.x-win-aarch64.exe`)
- Windows Intel 64-bit (`suzaku-x.x.x-win-x64.exe`)
- Windows Intel 32-bit (`suzaku-x.x.x-win-x86.exe`)

> [لسبب ما لا يعمل الملف الثنائي Linux ARM MUSL بشكل صحيح](https://github.com/Yamato-Security/hayabusa/issues/1332) لذلك فإننا لا نوفّر ذلك الملف الثنائي. الأمر خارج عن سيطرتنا، لذا نخطط لتوفيره في المستقبل عندما يتم إصلاحه.
