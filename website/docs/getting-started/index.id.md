# Unduhan

Silakan unduh versi stabil terbaru dari Suzaku dengan biner terkompilasi atau kompilasi kode sumber dari halaman [Releases](https://github.com/Yamato-Security/suzaku/releases).

Kami menyediakan biner untuk arsitektur berikut:
- Linux ARM 64-bit GNU (`suzaku-x.x.x-lin-aarch64-gnu`)
- Linux Intel 64-bit GNU (`suzaku-x.x.x-lin-x64-gnu`)
- Linux Intel 64-bit MUSL (`suzaku-x.x.x-lin-x64-musl`)
- macOS ARM 64-bit (`suzaku-x.x.x-mac-aarch64`)
- macOS Intel 64-bit (`suzaku-x.x.x-mac-x64`)
- Windows ARM 64-bit (`suzaku-x.x.x-win-aarch64.exe`)
- Windows Intel 64-bit (`suzaku-x.x.x-win-x64.exe`)
- Windows Intel 32-bit (`suzaku-x.x.x-win-x86.exe`)

> [Untuk alasan tertentu biner Linux ARM MUSL tidak berjalan dengan baik](https://github.com/Yamato-Security/hayabusa/issues/1332) sehingga kami tidak menyediakan biner tersebut. Hal ini di luar kendali kami, jadi kami berencana menyediakannya di masa mendatang ketika sudah diperbaiki.
