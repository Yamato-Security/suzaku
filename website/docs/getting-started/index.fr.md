# Téléchargements

Veuillez télécharger la dernière version stable de Suzaku avec les binaires compilés ou compiler le code source depuis la page [Releases](https://github.com/Yamato-Security/suzaku/releases).

Nous fournissons des binaires pour les architectures suivantes :
- Linux ARM 64-bit GNU (`suzaku-x.x.x-lin-aarch64-gnu`)
- Linux Intel 64-bit GNU (`suzaku-x.x.x-lin-x64-gnu`)
- Linux Intel 64-bit MUSL (`suzaku-x.x.x-lin-x64-musl`)
- macOS ARM 64-bit (`suzaku-x.x.x-mac-aarch64`)
- macOS Intel 64-bit (`suzaku-x.x.x-mac-x64`)
- Windows ARM 64-bit (`suzaku-x.x.x-win-aarch64.exe`)
- Windows Intel 64-bit (`suzaku-x.x.x-win-x64.exe`)
- Windows Intel 32-bit (`suzaku-x.x.x-win-x86.exe`)

> [Pour une raison inconnue, le binaire Linux ARM MUSL ne fonctionne pas correctement](https://github.com/Yamato-Security/hayabusa/issues/1332), c'est pourquoi nous ne fournissons pas ce binaire. Cela échappe à notre contrôle, nous prévoyons donc de le fournir à l'avenir une fois le problème corrigé.
