# ဒေါင်းလုဒ်များ

Suzaku ၏ နောက်ဆုံးတည်ငြိမ်သော ဗားရှင်းကို compile လုပ်ထားသော binaries ဖြင့် ဒေါင်းလုဒ်လုပ်ပါ သို့မဟုတ် [Releases](https://github.com/Yamato-Security/suzaku/releases) စာမျက်နှာမှ source code ကို compile လုပ်ပါ။

အောက်ပါ architectures များအတွက် binaries များကို ကျွန်ုပ်တို့ ပံ့ပိုးပေးပါသည်-
- Linux ARM 64-bit GNU (`suzaku-x.x.x-lin-aarch64-gnu`)
- Linux Intel 64-bit GNU (`suzaku-x.x.x-lin-x64-gnu`)
- Linux Intel 64-bit MUSL (`suzaku-x.x.x-lin-x64-musl`)
- macOS ARM 64-bit (`suzaku-x.x.x-mac-aarch64`)
- macOS Intel 64-bit (`suzaku-x.x.x-mac-x64`)
- Windows ARM 64-bit (`suzaku-x.x.x-win-aarch64.exe`)
- Windows Intel 64-bit (`suzaku-x.x.x-win-x64.exe`)
- Windows Intel 32-bit (`suzaku-x.x.x-win-x86.exe`)

> [အကြောင်းတစ်ခုခုကြောင့် Linux ARM MUSL binary သည် မှန်ကန်စွာ အလုပ်မလုပ်ပါ](https://github.com/Yamato-Security/hayabusa/issues/1332) ထို့ကြောင့် ထို binary ကို ကျွန်ုပ်တို့ မပံ့ပိုးပါ။ ၎င်းသည် ကျွန်ုပ်တို့၏ ထိန်းချုပ်မှုအပြင်ဘက်တွင် ရှိသောကြောင့် ပြုပြင်ပြီးသည့်အခါ အနာဂတ်တွင် ၎င်းကို ပံ့ပိုးပေးရန် စီစဉ်ထားပါသည်။
