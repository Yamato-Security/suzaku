# Downloads

Faça o download da versão estável mais recente do Suzaku com os binários compilados ou compile o código-fonte a partir da página de [Releases](https://github.com/Yamato-Security/suzaku/releases).

Fornecemos binários para as seguintes arquiteturas:
- Linux ARM 64-bit GNU (`suzaku-x.x.x-lin-aarch64-gnu`)
- Linux Intel 64-bit GNU (`suzaku-x.x.x-lin-x64-gnu`)
- Linux Intel 64-bit MUSL (`suzaku-x.x.x-lin-x64-musl`)
- macOS ARM 64-bit (`suzaku-x.x.x-mac-aarch64`)
- macOS Intel 64-bit (`suzaku-x.x.x-mac-x64`)
- Windows ARM 64-bit (`suzaku-x.x.x-win-aarch64.exe`)
- Windows Intel 64-bit (`suzaku-x.x.x-win-x64.exe`)
- Windows Intel 32-bit (`suzaku-x.x.x-win-x86.exe`)

> [Por algum motivo, o binário Linux ARM MUSL não é executado corretamente](https://github.com/Yamato-Security/hayabusa/issues/1332), por isso não fornecemos esse binário. Isso está fora do nosso controle, então planejamos fornecê-lo no futuro, quando for corrigido.
