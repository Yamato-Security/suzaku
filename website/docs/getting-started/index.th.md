# การดาวน์โหลด

โปรดดาวน์โหลด Suzaku เวอร์ชันเสถียรล่าสุดในรูปแบบไบนารีที่คอมไพล์แล้ว หรือคอมไพล์ซอร์สโค้ดจากหน้า [Releases](https://github.com/Yamato-Security/suzaku/releases)

เราจัดเตรียมไบนารีสำหรับสถาปัตยกรรมต่อไปนี้:
- Linux ARM 64-bit GNU (`suzaku-x.x.x-lin-aarch64-gnu`)
- Linux Intel 64-bit GNU (`suzaku-x.x.x-lin-x64-gnu`)
- Linux Intel 64-bit MUSL (`suzaku-x.x.x-lin-x64-musl`)
- macOS ARM 64-bit (`suzaku-x.x.x-mac-aarch64`)
- macOS Intel 64-bit (`suzaku-x.x.x-mac-x64`)
- Windows ARM 64-bit (`suzaku-x.x.x-win-aarch64.exe`)
- Windows Intel 64-bit (`suzaku-x.x.x-win-x64.exe`)
- Windows Intel 32-bit (`suzaku-x.x.x-win-x86.exe`)

> [ด้วยเหตุผลบางประการ ไบนารี Linux ARM MUSL ไม่สามารถทำงานได้อย่างถูกต้อง](https://github.com/Yamato-Security/hayabusa/issues/1332) ดังนั้นเราจึงไม่จัดเตรียมไบนารีนั้น เรื่องนี้อยู่นอกเหนือการควบคุมของเรา เราจึงวางแผนที่จะจัดเตรียมไบนารีนั้นในอนาคตเมื่อปัญหาได้รับการแก้ไขแล้ว
